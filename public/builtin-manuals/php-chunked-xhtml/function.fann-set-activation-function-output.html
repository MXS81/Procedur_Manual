fann_set_activation_function_output

(PECL fann >= 1.0.0)

fann_set_activation_function_output — 为输出层设置激活函数

说明

fann_set_activation_function_output(resource $ann, int $activation_function): bool

为输出层设置激活函数。

参数

ann

神经网络 资源。

activation_function

激活函数 常量。

返回值

成功时返回 true，其它情况下返回 false。
