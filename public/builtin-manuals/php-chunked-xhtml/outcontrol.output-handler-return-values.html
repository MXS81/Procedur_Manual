输出处理程序的返回值

输出处理程序的返回值在内部转换为字符串，遵循标准 PHP 类型语义， 但有两个例外：arrays 和 booleans。

Arrays 被转换为字符串 "Array"， 但不会触发 Array to string conversion 警告。

如果处理程序返回 false 则返回缓冲区的内容。 如果处理程序返回 true 则返回一个空字符串。

注意: 如果处理程序返回 false 或抛出异常，则设置其 PHP_OUTPUT_HANDLER_DISABLED 状态标志。
