imagegammacorrect

(PHP 4, PHP 5, PHP 7, PHP 8)

imagegammacorrect — 对 GD 图像应用伽玛校正

说明

imagegammacorrect(GdImage $image, float $input_gamma, float $output_gamma): true

在指定输入/输出伽玛的情况下，对给指定的 gd image 应用伽玛校正。

参数

image

由图象创建函数(例如imagecreatetruecolor())返回的 GdImage 对象。

input_gamma

输入伽玛。

output_gamma

输出伽玛。

返回值

总是返回 true。

更新日志

版本 说明
8.0.0 image 现在需要 GdImage 实例；之前需要有效的 gd resource。

示例

示例 #1 imagegammacorrect() 用法

<?php
// 创建图像实例
$im = imagecreatefromgif('php.gif');

// 校正伽马，输出 = 1.537
imagegammacorrect($im, 1.0, 1.537);

// 保存
imagegif($im, './php_gamma_corrected.gif');
?>
