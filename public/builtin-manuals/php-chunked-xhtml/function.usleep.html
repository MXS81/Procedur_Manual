usleep

(PHP 4, PHP 5, PHP 7, PHP 8)

usleep — 以指定的微秒数延迟执行

说明

usleep(int $microseconds): void

以指定的微秒数延缓程序的执行。

参数

microseconds

暂停的时间以微秒计。1微秒（micro second）是百万分之一秒。

注意: 操作系统不支持大于 1000000 的值（即休眠不能超过 1 秒）。使用 sleep() 代替。

注意: 任何系统活动或处理调用所花费的时间或系统计时器的粒度都可能会稍微延长睡眠时间（即可能会长于 microseconds）。

返回值

没有返回值。

示例

示例 #1 usleep()例子

<?php

// Current time
echo (new DateTime('now'))->format('H:i:s.v'), "\n";

// wait for 2 milliseconds
usleep(2000);

// back!
echo (new DateTime('now'))->format('H:i:s.v'), "\n";

// wait for 30 milliseconds
usleep(30000);

// back again!
echo (new DateTime('now'))->format('H:i:s.v'), "\n";

?>

以上示例会输出：

11:13:28.005
11:13:28.007
11:13:28.037

参见

  • sleep() - 延缓执行
  • time_nanosleep() - 延缓执行若干秒和纳秒
  • time_sleep_until() - 使脚本睡眠到指定的时间为止
  • set_time_limit() - 设置脚本最大执行时间
