pg_num_rows

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

pg_num_rows — 返回结果中行的数量

说明

pg_num_rows(PgSql\Result $result): int

pg_num_rows() 返回 PgSql\Result 实例中的行的数量。

注意:

本函数以前的名字为 pg_numrows()。

参数

result

PgSql\Result 实例，由 pg_query()、pg_query_params() 或者 pg_execute()（等）返回。

返回值

结果中行的数量。错误时返回 -1。

更新日志

版本 说明
8.1.0 现在 result 参数接受 PgSql\Result 实例，之前接受 resource。

示例

示例 #1 pg_num_rows() 示例

<?php
$result = pg_query($conn, "SELECT 1");

$rows = pg_num_rows($result);

echo $rows . " row(s) returned.\n";
?>

以上示例会输出：

1 row(s) returned.

参见

  • pg_num_fields() - 返回结果中字段的数量
  • pg_affected_rows() - 返回受影响的记录数（元组）
