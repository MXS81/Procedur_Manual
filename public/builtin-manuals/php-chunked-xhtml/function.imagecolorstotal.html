imagecolorstotal

(PHP 4, PHP 5, PHP 7, PHP 8)

imagecolorstotal — 取得图像调色板中的颜色数量

说明

imagecolorstotal(GdImage $image): int

返回图像调色板中的颜色数量。

参数

image

由图象创建函数(例如imagecreatetruecolor())返回的 GdImage 对象。

返回值

返回指定图像调色板中的颜色数量，真彩色图像为 0。

更新日志

版本 说明
8.0.0 image 现在需要 GdImage 实例；之前需要有效的 gd resource。

示例

示例 #1 使用 imagecolorstotal() 获取图像中的颜色总数

<?php
// 创建图像实例
$im = imagecreatefromgif('php.gif');

echo 'Total colors in image: ' . imagecolorstotal($im);
?>

以上示例的输出类似于：

Total colors in image: 128

  • imagecolorsforindex() - 获取索引的颜色
  • imageistruecolor() - 检查图像是否为真彩色图像
