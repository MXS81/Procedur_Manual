JavaScript 对象表示法（JSON）

简介

本扩展实现了 » JavaScript 对象表示法（JSON）数据转换格式。PHP 中的解析器，专门为 PHP 订制，软件许可证为 PHP license。

注意:

PHP 实现了 JSON 的一个超集，参考 » RFC 7159.
