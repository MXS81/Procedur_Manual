PHP 手册

关于本手册

PHP，即 PHP: Hypertext Preprocessor，是一种被广泛应用的开源通用脚本语言，尤其适用于 Web 开发并可嵌入 HTML 中去。它的语法借鉴了 C、Java 和 Perl，易于学习。该语言的主要目标是允许 web 开发人员快速编写动态生成的 web 页面，但 PHP 的用途远不只于此。

本手册内容主要由函数参考构成，但也包含了语言参考，PHP 一些主要产品特点的说明以及其它补充信息。

这份手册涵盖了 PHP 最近的两个主要版本 (7 和 8) 的相关信息。手册中只包含与 PHP 版本捆绑的扩展信息，并且不包含函数早期版本的更改信息。有关早期 PHP 版本的存档手册，可以在 » PHP 网站“Documentation”页面的“More documentation”章节找到。

可在 » https://www.php.net/download-docs.php 下载此手册的各种格式。更多关于如何开发本手册的信息可参阅附录：“关于本手册”。如果你对 PHP 的历史感兴趣，可访问相关附录。
